This app allows user to view all the blogs created so far on the website.
It allows user to sign up and login and the email id must be unique.
It allows user to create, edit and delete blogs only created by themselves.
The blogs created by current users can only be viewed on the home page by other users.
New users can also view all the blogs created by other users on the home page.
There are seprate buttons in navbar for each desired task such as: login, signup, create blog, show user's own blogs and show all blogs ever created.
By clicking on a desired blog user can either edit it or delete the blog only created by themselves.
Users can also change their email and password as well.

To start project run command: npm run dev
